## 🎯 Our Mission

> *"No patient should ever be overcharged for healthcare without knowing it."*

We built **Sanjeevani** because we believe healthcare billing should be transparent, fair, and accessible to every Indian citizen. Our platform uses AI to fight medical billing fraud — one bill at a time.

## 🏗️ How We Built It

### The Journey

```
Day 1  ─────────────────────────────────────────────
  ✦ Problem identification & research
  ✦ Architecture design & tech stack selection
  ✦ CGHS rate data sourcing & ingestion

Day 2  ─────────────────────────────────────────────
  ✦ OCR pipeline development (EasyOCR + Tesseract)
  ✦ Backend API & audit engine
  ✦ Frontend dashboard & upload flow

Day 3  ─────────────────────────────────────────────
  ✦ CGHS dual-source matching integration
  ✦ Deployment to Vercel + Render + HF Spaces
  ✦ Testing, bug fixes & documentation
```

### Our Values

| Value | Description |
|:---|:---|
| 🫀 **Patient First** | Every decision optimizes for the patient's benefit |
| 🔍 **Transparency** | Open source, clear methodology, auditable results |
| 🎯 **Accuracy** | 99% matching confidence — no false accusations |
| ⚡ **Speed** | Bill analysis in under 15 seconds |
| 🔒 **Privacy** | No bill data stored permanently — process and discard |
